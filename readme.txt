【タイトル】
RPG風バトルゲーム ～アイテムとスキルを駆使して敵を倒せ！～

【ファイル構成】
- main.cpp：ゲーム本体のソースコード
- readme.txt：このファイル

【概要】
C++で作成したテキストベースのバトルゲームです。
プレイヤーは敵と順番に戦いながら、スキルやアイテムを駆使して勝利を目指します。

バトルに勝つと敵からアイテムを入手でき、食べ物（Food）でHP/MPを回復したり、
武器（Weapon）を装備して攻撃力を強化することができます。

【使用方法】
1. ターミナルで `g++ main.cpp -o main.exe` を実行してビルド。
2. `./main.exe` を実行。
3. 指示に従って入力を進めてください。

【使い方例】
- 名前を入力
- 敵と戦闘（攻撃 / スキル / インベントリ）
- インベントリから食べ物を使用するとHP/MPが回復します
- 武器を選択すると攻撃力が変わります
- 食べ物を使うと自動的に持ち物から削除されます

【特徴・工夫点】
- STLの `vector`, `remove_if`, `erase` を使用したアイテムの管理
- 動的キャスト (`dynamic_cast`) によるアイテムの分類（武器・食べ物・素材）
- 仮想関数による柔軟なクラス設計
- 敵キャラクターはスキルを使ってくるAIを搭載
- 1つの `.cpp` ファイルにすべてをまとめて構成

【使用したSTLコンテナとアルゴリズム】
- `vector<Item*> inventory`：アイテム一覧の管理
- `remove_if` + `erase`：使用済みアイテムを削除
- `dynamic_cast`：ポリモーフィズムを活かしたアイテム抽出
- `iterator`：STLと連携したループ処理

【動作環境】
- C++11以降
- MinGW64, g++ コンパイラ推奨

class ItemManager {
    static int count;      // static変数
    int instanceID;        // 非static変数

    static void staticFunc() {
        // countはOK
        // instanceIDはNG（thisがないので）
    }

    void nonStaticFunc() {
        // 両方アクセスOK
    }
};
のような構造があった場合、

staticFuncからはcountだけ直接触れて、instanceIDはアクセスできません。

nonStaticFuncはthisがあるので両方触れます。
